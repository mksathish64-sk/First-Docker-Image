# Hangman Game using Python

This is a simple Hangman game built using Python.  
The project is Dockerized to understand the basics of Docker, image creation, and pushing images to Docker Hub.

## Technologies Used
- Python
- Docker

## Run Locally
```bash
python hangman.py
```

## Run Using Docker
```bash
docker run -it --name word-game mksathish64/hangman-game:v1
```

## What I Learned
- Basics of Docker and containerization
- Writing a Dockerfile
- Building Docker images
- Running containers
- Pushing images to Docker Hub
- Containerizing a Python application
